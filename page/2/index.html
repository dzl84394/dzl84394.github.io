安装 Hexo

官方文档
https://hexo.io/zh-cn/docs/setup

  • 升级npm
    npm install npm -g
  • 安装
    npm install -g hexo-cli

安装 Hexo 完成后，请执行下列命令，Hexo 将会在指定文件夹中新建所需要的文件。

$ hexo init
$ cd
$ npm install

添加一篇文章
$ hexo new ‘hello word’

生成静态文件
$ hexo generate
可以简写：
$ hexo g

启服务
$ hexo server

部署网站
$ hexo deploy

npm install hexo-deployer-git –save

hexo g -d

学习Jekyll

安装ruby

不知道为啥我mac自带的ruby安装jekyll总是无法成功，所以只能另外自己重新安装一个
先安装rvm：

  • 下载&安装
    curl -sSL https://get.rvm.io | bash -s stable
  • 环境变量
    source /Users/zlding/.rvm/scripts/rvm
  • 查看版本
    rvm -v
  • 更新rvm
    rvm get stable

  • 版本查看
    rvm list known

  • 安装一个版本
    rvm install 2.5.3
    rvm use 2.5.3 –default

安装Xcode

要Xcode/Command Line Tools
如果下载不了
https://developer.apple.com/download/more/
去下载，自己安装

安装jekyll

rvm装好了，选一个ruby，gem也好了

gem -v
gem update –system

gem install jekyll
jekyll -v
gem install bundler
gem install jekyll-paginate
gem install jekyll-gist
