学习Jekyll

安装ruby

不知道为啥我mac自带的ruby安装jekyll总是无法成功，所以只能另外自己重新安装一个
先安装rvm：

  • 下载&安装
    curl -sSL https://get.rvm.io | bash -s stable
  • 环境变量
    source /Users/zlding/.rvm/scripts/rvm
  • 查看版本
    rvm -v
  • 更新rvm
    rvm get stable

  • 版本查看
    rvm list known

  • 安装一个版本
    rvm install 2.5.3
    rvm use 2.5.3 –default

安装Xcode

要Xcode/Command Line Tools
如果下载不了
https://developer.apple.com/download/more/
去下载，自己安装

安装jekyll

rvm装好了，选一个ruby，gem也好了

gem -v
gem update –system

github创建一个项目

github建立一个和用户一摸一样的项目
dzl84394.github.io
username必须一样
后面的.github.io会默认这是一个博客

在_post文件夹添加mk文件

_site 文件夹上传到dzl84394.github.io.git，这边是网站静态文件。
网上一直忽略这一点，直接把源文件上传上去，这很坑的
