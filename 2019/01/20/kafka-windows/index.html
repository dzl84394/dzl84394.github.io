kafka window 入门

kafka windows环境

https://blog.csdn.net/u010054969/article/details/70241478

  • 下载
    http://kafka.apache.org/downloads.html
    不要下源码，下tar包的

  • 解压之后
    里面有zk，有kafka，不知道为啥外面的zk起会报错，自带完全没问题

  • 报错：找不好主类的错误
    bin\windows\kafka-run-class.bat的179行
    “%CLASSPATH%” 打上双引号

  • 修改配置
    打开D:\kafka_2.11-0.10.2.0\config\zookeeper.properties
    dataDir=D:\logs\zookeeper
    打开D:\kafka_2.11-0.10.2.0\config\server.properties
    log.dirs=D:\logs\kafka
