Eclipse PlantUML

1,sublime+PlantUML

  • 下载安装 graphviz
    http://www.graphviz.org/download/
    点击 Stable 2.38 Windows install packages
    下载zip版，
    解压，添加环境变量
    变量名：PATH
    变量值：D:\graphviz-2.38\bin
    变量名：GRAPHVIZ_DOT
    变量值：D:\graphviz-2.38\bin\dot.exe

  • 下载PlantUML插件
    https://github.com/jvantuyl/sublime_diagram_plugin
    解压之后，放在sublime的Packages路径下

通过 Preferences -> Browse Packages … 打开 sublime 的 Packages 目录

  • 快捷键
    为了简化使用，可以在 Sublime 里配置个快捷键。打开 Preferences -> Key Binding - User，添加一个快捷键

上面的代码配置成按住 Alt + d 来生成 PlantUML 图片，你可以修改成你自己喜欢的按键。
