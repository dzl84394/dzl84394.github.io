mac日常操作

1.多开sts或eclipse

2.日常软件启动

  • mongo
    sudo ./bin/mongod

  • kafka &zk
    sudo ./zookeeper-server-start.sh ../config/zookeeper.properties
    sudo ./kafka-server-start.sh ../config/server.properties

  • redis
    cd /usr/local/redis/
    ./src/redis-server

3.安装Navicat Premium for Mac

网上的教程：https://www.jianshu.com/p/f3ef78deadaa

有2个坑
1，就是rpk文件，最新版已经没有了，这个路子走不通了，得下个12.0.22之前的
2，教程里面的rsa的 http://tool.chacuo.net/cryptrsaprikey 加密解密网站，在移动网站是打不开的，电信没问题

SecureCRT

1，防止中断
公司搞了跳板机，堡垒机一类的，导致很容易掉线，堡垒机又不记录密码，坑！
【terminal】 下 【send string】设置每60秒发一个空格，可以防止中断
2，记住密码
默认的mac版是不记录密码的，菜单preferences–general，找到mac options。然后去掉Use KeyChain选项，可以让记住密码，
