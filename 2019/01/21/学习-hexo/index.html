安装 Hexo

官方文档
https://hexo.io/zh-cn/docs/setup

  • 升级npm
    npm install npm -g
  • 安装
    npm install -g hexo-cli

安装 Hexo 完成后，请执行下列命令，Hexo 将会在指定文件夹中新建所需要的文件。

$ hexo init
$ cd
$ npm install

添加一篇文章
$ hexo new ‘hello word’

生成静态文件
$ hexo generate
可以简写：
$ hexo g

启服务
$ hexo server

部署网站
$ hexo deploy

npm install hexo-deployer-git –save

hexo g -d
